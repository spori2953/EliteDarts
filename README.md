# \# EliteDarts

###### 

###### Web-based darts tournament management system with computer vision throw detection.

###### 

###### \## Technologies

###### \- ASP.NET Core

###### \- Blazor Server

###### \- Entity Framework Core

###### \- SQL Server LocalDB

###### \- .NET 8

###### \- OpenCvSharp

###### 

###### \## Projects

###### \- `EliteDarts` – web application

###### \- `EliteDarts.CvWorker` – computer vision microservice

###### \- `EliteDarts.CvWorker.Tests` – automated tests

###### 

###### \## Requirements

###### \- Visual Studio 2022

###### \- .NET 8 SDK

###### \- SQL Server LocalDB

###### 

###### \## How to run

###### 

###### 1\. Clone the repository

###### 2\. Open `EliteDarts.sln`

###### 3\. Restore NuGet packages

###### 4\. Start `EliteDarts.CvWorker`

###### 5\. Start `EliteDarts`

###### 

###### \## Tests

###### Run the tests from Visual Studio Test Explorer or with:

###### 

###### ```bash

###### dotnet test```



###### \##Notes

###### 

###### The system includes mock-based automated tests for the computer vision module.
